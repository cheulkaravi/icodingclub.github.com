HTML5 Canvas Image cropper: Flexible selection approch

Introduction


This is a client side only image cropper, built out of HTML5 or HTML5 Canvas. Code is object oriented and bulit purely in Javascript without use of any library. cropping can be done in flexible manner by selecting any region on UI, by mouse.

DEMO


How to run this demo?

  1. Select a area using mouse.
  2. Click on crop.
